Omega Nu Turn Based Game Engine

This was a project I worked on with nine other people. It's a game engine that allows you to create your own turn based RPGs and play them! Inspired by Fire Emblem and other similar games, the schema for generating content is extremely flexible. The code is hosted here on Github. It was built in mostly Java, with JSON serving as the format in which we stored saved game modes and games.
